Badge levels¶

Four statuses exist for the Diversity and Inclusion Event Badging:

| Level | Badge | Percentage of Requirements Met | | :— | :— | :— | | Pending | | Less than 40% | | Passing | | Greater than or equal to 40% and less than 60% | | Silver | | Greater than or equal to 60% and less than 80% | | Gold | | Greater than 80% |
